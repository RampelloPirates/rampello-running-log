Import a Garmin run (FIT)

Upload one activity's .fit (or .fit.gz) file. It's decoded in your browser, and each Garmin lap becomes a run segment with its own measured cadence and heart rate — so you can see how those shift across the run.

Where to get the FIT file
  1. From Garmin Connect (one run): open the activity → gear/⚙ menu → Export Original → you get a .fit.
  2. From your Strava export: inside the ZIP, the activities/ folder has the original files as .fit.gz — upload one of those.
